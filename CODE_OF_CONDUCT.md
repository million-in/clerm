# Code of Conduct

## Our Standard

We expect contributors to keep technical discussions direct, respectful, and grounded in evidence. Disagreement is acceptable. Personal attacks, harassment, discrimination, and deliberately disruptive behavior are not.

## Expected Behavior

1. Focus on the technical problem.
2. Critique ideas with specific reasoning.
3. Avoid hostility, abuse, and intimidation.
4. Respect project boundaries, especially between the public compiler and the private registry service.

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict participation for conduct that violates this policy.
